## h2 Question ->

**Write a JavaScript function that reverse a number. Sample Data and output: Example x = 32243; Expected Output : 34223**

```HTML
 <body onload="reverse()"">
    <script>
      function reverse() {
        var r = prompt("enter the number");
        var t = r.split("").reverse().join("");
        document.write(t);
      }
    </script>
  </body>
```
